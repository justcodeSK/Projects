# Threat Intelligence, Web Security & Network Analysis

## Overview

This project focuses on practical cybersecurity analysis involving threat intelligence, web security assessment, and network traffic analysis. It combines OSINT techniques, packet analysis, and web vulnerability testing to understand real-world attack vectors and defensive security mechanisms.

## Tools & Technologies

* Wireshark
* Nmap
* VirusTotal
* URLScan
* MXToolbox
* Burp Suite
* PortSwigger Web Security Academy
* Kali Linux

## Key Activities

### 1. Threat Intelligence & Phishing Analysis

* Investigated phishing emails using header analysis and IOC extraction
* Performed OSINT-based analysis of malicious domains, IPs, and URLs
* Verified threat indicators using tools like VirusTotal and URLScan

### 2. Network Traffic Analysis

* Captured and analyzed network packets using Wireshark
* Conducted port scanning and service enumeration using Nmap
* Identified open ports, services, and potential misconfigurations

### 3. Web Security Assessment

* Performed vulnerability testing on web applications
* Completed hands-on labs covering SQL Injection (SQLi) and Cross-Site Scripting (XSS)
* Practiced exploitation and detection techniques in controlled environments
